# AssesHub - Online Assessment Platform

Live Demo:
https://LikshaniPriya.pythonanywhere.com

Project Description:
A Django-based online assessment platform with role-based authentication, assessment creation, submission management, and malpractice detection features.

Source Code:
Due to the project structure and large number of files, the complete source code is provided as a downloadable ZIP archive in this repository.

Instructions:
1. Download the ZIP file.
2. Extract the project folder.
3. Install dependencies.
4. Run the Django application locally.

Tech Stack:
- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

Features:
- Student and Instructor roles
- Assessment creation
- MCQ and descriptive questions
- Submission tracking
- Activity logging
- Fullscreen monitoring
- Tab-switch detection
